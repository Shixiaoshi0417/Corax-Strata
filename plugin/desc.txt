🦅 墨鸦-Strata v4.4.0 🦅

"墨鸦"——鸦科记忆力冠绝自然，让 AI 像乌鸦一样记住你。

项目链接: https://github.com/Shixiaoshi0417/Corax-Strata

初次使用：
/ai set api_key <你的 sk-xxx key>
/ai set search_api_key <你的搜索 key>
/ai on 启用当前对话框 AI 功能

近期更新（v4.4.0）：
- 🎙️ Edge TTS 语音合成：/ai tts on/off 开关，/ai tts voice <name> 切换音色
- 🔊 7 个验证可用音色：晓晓/晓伊/晓萱（女）云希/云健/云扬/云夏（男）
- 💬 TTS 模式下文本与语音同时输出（文字 + 语音）
- 🧹 精简代码，移除未使用的 SILK/AMR 编码逻辑
- 🐛 修复 setAiConfig 误删、BeanShell 解析错误等多个 Bug

默认值（可用 /ai set 更改）：
model = deepseek-v4-flash
context_ttl = 60（对话保留一小时）
max_turns = 60（保留最新60轮）
ai_url = https://api.deepseek.com
search_provider = tavily（同时支持博查＆Bing）
show_stats = 0
debug = 0
temperature = 0.7
pat_wake = 1（拍一拍唤醒）
ai_prefix = 1（AI 消息强制 [AI] 前缀）
search_rounds = 3
sewarden = 1（WARDEN增强防护，不建议关闭）
tts_voice = zh-CN-XiaoxiaoNeural（TTS默认音色）

核心架构：
1. Strata 热冷分层记忆
2. CAST 三段式可信度
3. WARDEN-I 三层身份隔离 + SEWarden 标签逃逸防护
4. STREAM 稳定上下文 + 监听唤醒
5. DREX 格式化执行器
6. 全尖括号标签协议，QQ 昵称天然防御
